Проект на Flutter с минимальной версией платформы iOS 14.0.
В pubspec.yaml используются плагины:  
- firebase_core  
- firebase_remote_config  
- google_sign_in  

Сейчас CocoaPods не может собрать проект для iOS из-за конфликта версий пода "GoogleUtilities/Environment":  
— firebase_core требует ~> 7.12  
— firebase_remote_config требует ~> 7.8  
— google_sign_in через GoogleSignIn тянет ~> 8.0

Задача:  
1. Подбери такие конкретные версии пакетов firebase_core, firebase_remote_config и google_sign_in для pubspec.yaml, чтобы не было конфликта по GoogleUtilities/Environment и сборка iOS проходила.
2. Приведи итоговый пример секции dependencies для pubspec.yaml.
3. Кратко объясни порядок действий по обновлению (flutter pub get, pod install и т.п.).
4. (Опционально: если это невозможно в текущих версиях, укажи альтернативы или временный обходной путь).
